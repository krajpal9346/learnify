# Learnify

An AI-powered learning platform designed to help students discover courses, track progress, and enhance their skills.

## Features
- Authentication
- Personalized dashboard
- Course recommendations
- Progress tracking
- Responsive UI

## Tech Stack
- React
- Vite
- Tailwind CSS
- React Router
- Lucide Icons

## Installation

npm install
npm run dev
